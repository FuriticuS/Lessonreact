1. NodeJs - программная платформа (вспомогательные пакеты помогают писать js для браузера)

2. SPA - одностраничное приложение 

3. Функциональная компонента - функция js (компонента без состояния)

4. props - объект (который можно получить с атрибута тега=компоненты, каждое название атрибута = свойство объекта props, значение атрибута = значение объекта props)

5. map - функция js (метод массива = получает новый массив из массива )

6. Route - отрисовка страницы по адресу URLа (следит за адресной строкой браузера)

7. UI - BLL (react=user interface - redux=business logic layer)
